Commands to push to GitHub:
============================

1. First, create the repository on GitHub:
   - Go to https://github.com/new
   - Repository name: games
   - Make it Public
   - Don't initialize with README
   - Click "Create repository"

2. Then run these commands from the games directory:

cd C:\Users\bk\mini_zen\games

git branch -M main
git remote add origin https://github.com/brikaye/games.git
git push -u origin main

3. Enable GitHub Pages:
   - Go to repository Settings â†’ Pages
   - Source: Deploy from branch "main"
   - Folder: / (root)
   - Save

4. Your games will be live at:
   https://brikaye.github.io/games/
